## 📘 Overview

**Login-with-Firebase** adalah proyek **Fullstack Authentication App** dengan sistem login manual dan login via Google (Firebase).  
Proyek ini mencakup frontend (React + TypeScript), backend (Express + PostgreSQL), dan sistem reset password melalui email (Resend).

---

## ⚙️ Tech Stack

| Layer | Teknologi |
|-------|------------|
| **Frontend** | React + TypeScript + Vite |
| **Backend** | Node.js + Express |
| **Database** | Neon PostgreSQL |
| **Auth** | Firebase Authentication + JWT |
| **Email** | Resend API |
| **Hosting** | Frontend: Vercel · Backend: Railway |
